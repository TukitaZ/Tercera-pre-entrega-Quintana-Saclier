En este proyecto web se simula un sistema de un colegio, con seccion home, alumnos, maestros, materias.

con las clases: materias(nombre, comision)
		maestros(nombre, apellido, documento, email, profesion)
		alumnos(nombre, apellido, documento, email, comision)

En la pagina se encuentran redirecciones por cada una de las clases. En las cuales se encuentran:
Visualizacion de base de datos.
Busqueda de cada tipo de clase.
Formularios para crear cada tipo de clase.

Cada clase contiene sus archivos html por separado. Como a su vez sus funciones.
